# EOSIO.CDT (Contract Development Toolkit)


## 笔记：

### Installation (Building from Scratch)
```sh
$ git clone --recursive https://github.com/eosio/eosio.cdt
$ cd eosio.cdt
$ ./build.sh
$ sudo ./install.sh
```
### Compile example hello

cd examples/hello
cmake . 
make
